Typical Workflow
  • irondome create bunker # one-time setup
Setup
  • Choose: Touch ID / Windows Hello / Password
  • Configure defaults (length, characters)
  • Save recovery key (shown once)
  • irondome open airspace # authenticate once
Airspace Open — 30 min
  • Biometric / password → free access for 30 minutes
  • bunker create # no re-auth needed
  • bunker create # still free
  • bunker open github # instant access
  • bunker fortify # encrypted backup
  • irondome close airspace # lock down
  • bunker open # blocked
  • Airspace is closed. Run 'irondome open airspace' first.

Defense Capabilities

Six layers of protection for your credentials

AES-256 Encryption

Fernet symmetric encryption with PBKDF2-HMAC-SHA256 and 600,000 iterations. OWASP 2023 compliant.

Zero Knowledge

Your master password is never stored. Only a salted hash exists. Not even the tool can read it.

Hardware Bound

Encryption keys derived from your machine's hardware identifiers. Data is tied to your device.

Brute Force Shield

Adaptive lockout system with progressive security. Each failed attempt tightens the defense.

Session Protection

Automatic timeout after 30 minutes of inactivity. Sensitive operations require re-authentication.

Local Only

No network. No cloud. No telemetry. Everything stays on your machine. Period.
